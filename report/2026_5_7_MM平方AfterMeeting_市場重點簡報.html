  • Copilot / GitHub Copilot 是 AI Agent 落地主力

Meta — 投入最積極，市場最存疑

  • Google CEO：算力供給仍緊張，供不應求狀態維持

AI Agent — 2026 最大關鍵字

從被動對話 → 主動完成任務

  • ChatGPT 解決了「AI 聊天」問題，下一步是 Agent
  • S&P 500 電話會議提及「AI Agent」次數急速攀升
  • Claude Code 年初爆發 → 文組也能生成程式
  • 各廠商都在推進 Agentic AI：
    • Google：Agentic Commerce（AI 幫你買東西）
    • Amazon：AI 代理購物 + ASIC
    • Meta：廣告 + 數位客服 engagement
    • Microsoft：Copilot / GitHub Agent
為何 Agent 更消耗算力？

Agent 需要多輪推論、工具調用、長上下文維護，
每次任務消耗的 Token 遠高於簡單對話，
直接帶動 GPU / 記憶體 / 電力需求再上一層樓。

「你可能半年前以為做不到的事情，AI 就可以幫你完全重新做出來，所以常常會有意想不到的驚喜。」

Jason 的三大投資趨勢觀察

① AI 基建潮：指數級上升
  • 四大 CSP CAPEX 持續上修
  • GPU / CPU / 記憶體 / 電力全面緊俏
  • Google CEO 親口說：算力供給仍是限制
  • OpenAI、Anthropic 都在搶租資料中心
  • 整個行業供需偏緊，遠未到供給過剩
② 競爭加速：分散佈局最優解
  • LLM 競爭洗牌週期縮短至數個月
  • 不需要單壓 GPU 陣營或 ASIC 陣營
  • 不需要單壓 Google 或 Microsoft 陣營
  • 若 OpenAI / Anthropic 上市也無需單壓
「小孩子才做選擇，我都要。」
③ 留意風險：現金流與債務
  • 五大雲端業者（4 大 CSP + Oracle）

另需關注總體風險：地緣政治升溫、通膨持續嚴峻、美國財政與貨幣政策時間點

量子運算 vs 矽基半導體 / 原油市場快速解析

量子運算

現況：互補而非替代
  • 適合場景：新藥計算、蛋白質結構、密碼學破解、複雜優化
  • 傳統 CPU/GPU 在確定性任務上成本效率仍勝出
  • 台積電、傳統半導體廠短期無被顛覆風險
風險點在加密 / 資安

量子破解密碼學的潛力，使「後量子加密」成為市場討論焦點（IONQ 等量子股）

原油

短期：霍爾木茲瓶頸仍緊
  • 中東透過霍爾木茲原本運輸量 2,000 萬桶/日
  • 阿聯酋管線：上限 150 萬桶/日，遠不足補缺口
  • 美國頁岩油：開採成本 ~66 美元，短期增產有限
  • 戰略儲油 SPR + 商業庫存：尚有 500–1,000 萬桶缺口
聖旨風險

「川普一則推特，油價直接跌 15 美元。」
玩不過川普，短線交易者需特別注意政策突發風險。
