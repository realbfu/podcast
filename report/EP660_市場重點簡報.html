大盤概況：強到不知道發生什麼事

市場數據
  • 台股大盤指數：42,000 點
  • 台積電股價：2,345 元
  • 台灣過去 10~20 年年化報酬：世界冠軍（20%+）
  • 第二名羅馬尼亞、第三名哈薩克
  • 五月第二週，漲勢沒有煞車跡象
市場氛圍
  • 身邊朋友幾乎沒人主動降槓桿
  • 盤中出現漲停打開再鎖回 → 籌碼洗乾淨
  • 成本拉太遠 → 回檔不痛，麻木
  • 回檔型態預期：賺太多 → 隨便亂買亂丟
  • 要讓投資人賠錢，難度極高
我已經不知道這個盤到底是發生什麼事情了 — 但身邊沒有多少人要提早去降槓桿。

NVIDIA 本體

  • 下一代可能再次採用相關材料
  • GTC 大會在即，主持人將出席並分享

現在 X 上有一堆動漫照片的分析師帳號，已把消息傳遍市場一圈 — 那種時候就是可以放心聊的時候。

消費性 / 成熟製程族群：悄悄轉強

世界先進（5347）
  • 成熟製程，產能賣光
  • 海外客戶願意出溢價搶產能，但拒絕簽約
  • 代表業者對未來行情信心強
  • 股價強勢表現
環球晶（6488）& 連電（2323）
  • 矽晶圓週期往上
  • 環球晶法說內容「時機未到」但股價強勢
  • 日本及其他矽晶圓相關也表現好
  • 連電強勢突破
Global Foundries（GFS）
  • 海外交易市場表現非常好
  • 成熟製程代工族群
  • 台灣 DDIC 業者曾提前拉貨
  • 消費性/車用相關成熟製程悄悄在動
是不是消費性復甦？主持人猜測更像是關稅 + 漲價 → 提前拉貨 — 下游怕斷供，代工廠產能利用率先拉起來，IC 設計 Q2 營收先跳，股價再往前跳。

被動元件：全面爆發，不只 MLCC

已動的族群
  • MLCC（陶瓷電容）— 已啟動
  • 鋁電容：Nichicon、Nippon Chemi-Con（日系強勢）
  • 美股 Vishay（VSH）、VPG — 表現很好
  • 電感、電阻 — 也可稍微關注
產業判斷
  • 原預期 2026 下半年～2027 才缺貨，提早動起來
  • 可能先動一小波 → 休息 → 再一波真正的大缺
  • 類似記憶體的「先動先休」模式
  • 需求極大，但供給端結構性問題仍在

類比過去

  • CPU 那波：先從技術面發現族群性啟動 → 再研究基本面驗證 → 布局
  • 記憶體：也是先看到股價跳，再確認漲價邏輯
  • 被動元件現在的族群性訊號 → 值得深入研究

台灣新一波財富重分配

現象觀察
  • 4 月台灣億萬富翁數量大幅增加，部分前十大大股東資產暴增數十%
  • 台積電等大型股大噴射 → 本來千萬的可能破億，破億的可能變十億
  • 不只前十大，前三十名、前五十名大股東每檔都是億萬富翁
  • 0050 & 6208 受益人極多 → 多數人其實都有參與到這波經濟成長
打不贏就加入 — 進不了台積電可以買台積電。文組宅也好，靠爸也好，現在透過投資直接參與整體經濟成長，這個時代比以前方便太多。

主持人提醒

  • 抱怨原生家庭、怨天尤人 → 幹了半天沒有意義
  • 要嘛加入他們（投資參與），要嘛繼續幹幹叫 — 就這兩個選項
  • 像台積電估值才 10~20 倍，真的很貴嗎？比較便宜的東西還是可以持續持有

資金面警訊：融資額度被 Cap

融資環境變化
  • 台股「晴天收傘」— 不讓做更多融資
  • 新開戶融資額度被鎖，連財力都無法解鎖
  • 抵押帳號利率從 2.x% 調升至 3%
  • 主持人自身：可以借更多，但券商說「我沒有錢借你」
  • 參與人數增加 → 額度被分光
主持人解讀
  • 不一定代表大家都把槓桿開到最高
  • 市值貴了一倍 → 同樣擔保品可借金額上限也相對被 cap
  • 對特定大戶：是被迫降槓桿，而非主動
  • 整體影響：主持人認為對市場影響有限
我等於被迫這樣杠杆 — 他把我的上限給 cap 掉了。不代表每個人都把杠杆開更高，只是進入一個更疯狂的局面被限制住。

大型股處置機制：法人開始抗議

聯發科（2454）
  • 連續漲停後進處置股
  • 流動性被抽掉引發討論
  • 外資法人打電話進去罵人
某散熱大廠（市值數千億）
  • NVIDIA 改料事件 → 連續跌停
  • 進處置後流動性完全喪失
  • 「關廁所」效應 — 想跑都跑不掉
  • 比賠錢更恐怖的事

市場討論方向

  • 大型股（幾千億市值）是否應免於 5% 漲跌停限制 / 處置制度
  • 政府可能修法：依市值大小決定是否適用處置機制
  • 美股機制：可以一天跌 30%，但至少任何時候都能出場
  • 台灣：越擁擠的股票 + 越多槓桿 → 連鎖跌停斷頭更恐怖

記憶體 & Tier1 → Tier2 輪動

記憶體現況
  • 美股記憶體：非常強勢
  • 台股記憶體：開始稍顯無力，可能只是籌碼問題
  • 類似過去金居（超大族群 → 籌碼太多 → 必須盤整）
  • 展望與目前狀況仍好，但股價已反映多少？
Tier1 → Tier2 輪動邏輯
  • Tier1（光通、記憶體等）開始軟掉或盤整
  • 資金轉向 Tier2（消費性、成熟製程、被動元件、工業電腦）
  • 工業電腦也在漲，PMIC 也在動
  • 沒有什麼大故事，可能只是資金找低估值目標
如果 Tier1 的東西開始軟掉，我可能就去找 Tier2 的東西 — 直到沒有東西可以找的話，就知道盤要進入休息了。

操作心態：進入「間末期」

主持人宣布「不回答個股問題」
  • 大盤在 42,000 點高位，不敢再幫人評論任何個股
  • 理由：你買了 → 後面繼續回檔 → 你每天跑來問我
  • 身邊朋友問股票 → 直接說：「你去問別人。」
自身部位做法
  • 若手上部位出現大逃殺訊號 → 即便故事不嚴重，仍小幅減碼
  • 不急著立刻補部位、抄底
  • 整體位置在高點 → 任何減碼都要考量全局
  • 等待反轉跡象：大盤出現崩殺訊號才降槓桿
減碼 vs 守倉
  • 成本距離太遠 → 老子直接不降（前集）
  • 但估值貴的東西 → 選擇繞開
  • 2028~2029 年 51 百倍 PE 的東西 = 主持人認為有泡沫感
  • 台積電等大權值股 10~20 倍 → 相對不貴

Q&A 精選

Q：CPU 類股已漲到看不懂，應否出場？
  • 主持人：不可能賣掉還在強勢進攻的東西
  • 站在賣方 = 開始找理由丟一點（跌破五日線、出黑 K）
  • 絕對不主動砍好的東西 — 違反所有邏輯
Q：資產配置 68.6% 股票 / 31.4% 現金，會不會太保守？
  • 每個人有自己的耐受度，不需要改變
  • 直接把現金全丟進去可能整晚睡不著
  • 知足常樂非常重要
  • 不用追求杠上加杠
買 ETF 賺錢快感遠低於個股？我還是會自己交易 — 研究產業好玩、認識交易圈朋友，賺錢最重要，但這個過程本身也有意思。
