台積電：劇本被外資打亂

原本的多頭劇本

期待台積電像聯發科一樣「動能股噴射」
大牛股突然起噴會非常恐怖

高盛喊 MediaTek $2,454、台積電 $2,330 目標價

外資的反手

外資不想讓台積電繼續往上衝

  • 連兩天賣超：1.8 萬張、2 萬多張
  • 最高摸到 2,330 後直接軟掉

主持人觀點

「兩個東西的上限應該都不是在這邊。要等外資的賣壓走到什麼時候再判斷。」

「垃圾盤」現象再現

規模太大 → 把市場資金全吸走

禮拜一：台積電強
  • 中小型直接殺到爆掉
  • 一堆光通訊跌停
  • 典型「垃圾盤」
禮拜二：台積電橫盤
  • 中小再次轉強復活
  • 資金重新流出來
台積電沒一波出去 → 後續資金會在台積電這邊還是中小？目前難判斷。

記憶體：軟很久後集體表態

旺宏法說會：老闆轉「超級利多」
  • 之前大家都說好的時候，老闆都不敢講好
  • 這次終於出來說自己很好 → 有人擔心是「議反應」訊號
  • 產品線實際上確實有漂亮漲價

基本面驗證

  • DRAM DDR3 / DDR4 漲價兇猛、Nor 也不錯
  • NAND MLC NAND 漲價、消費級 SSD 同步噴
  • 關鍵 「被排擠的標的」反而是漲價最猛的地方

被動元件：台股補漲海外

台股 underperform

禮拜二才趁機表態跳出去
之前已經落後海外蠻多

海外早就動起來
  • 日本幾大被動元件公司
  • 韓國 SAMCO 整個射出去
  • 漲價信一直跑出來，基本面驗證對
若覺得台股被動軟，現在仍可輕易買到日股 / 美股相關標的；台股說不定會在後面跟上。

本期關鍵：Intel EMIB 封裝良率 90%

後段封裝 (Assembly) 良率已拉到 90%
  • 跟台積電比仍偏差，但 90% 絕對可量產
  • 市場買方、賣方分析師多半還不知情或不確定
  • 主持人問完產業圈朋友也都「不太相信數字這麼高」

為何市場資訊落後？

  • 多數人從「載板端」 check → 載板電容良率僅 30–40%
  • 所以大家假設後段封裝也很差
  • 類比 Grace Blackwell 量產初期：代工廠看到的良率也只有 40–50%，後來慢慢 ramp 上去

Intel EMIB 的投資意涵

前段先修正預期

之前提過 Intel 若做不出來 → MTK 可能 BBQ

現在要修正：90% 良率代表後面只會更高，而且已可量產

把 Intel 視為「專業後段封裝廠」
  • 前段仍卡關（18A 短期 1–2 年困難）
  • 但後段封裝就有機會多拿訂單
  • Intel 高層的「底氣」可能來自這裡

是 TSMC / 後段封裝廠的利空嗎？

不是。馬斯克的說法正確：現在大家極度缺產能，只是排隊客人可能跑到 Intel；TSMC / 強大後段封裝廠仍會持續排隊。除非 AI 明顯反轉，否則「產能多少、就能投多少」。

EMIB 敘事帶起的個股

  • AP Memory 做 Intel 載板裡的 IPD（埋在載板裡的被動式電容）
  • MTK 本身強勢
  • 族群 跟 MTK / Google / Intel EMIB 相關標的多半都動起來
主持人的提醒

講這些時要強調「絕對不是高檔喊標題」——這些標的早就一路 cover、一路聊，要查就查最前面的節目。

小支的不做太多論述（避免被認為喊盤），大支的可以分享。

盤面總結：混沌局面

原本的劇本

  • 台積電一波打出去 → 啟動新一輪類股輪動

實際發展

  • 台積電摸 2,330 後就軟掉、櫃買在台積電轉強時就先做頭往下
  • 中小死一天又反彈，但反彈後有些東西轉弱
  • 反而是過去牛皮的 被動元件、記憶體 動起來
當前定調：肋骨帶開表現

不是看基本面（好東西很多），而是看資金往哪裡跑。
重點觀察：聯發科法說會是否給出更多提示。

OpenAI × 立訊：AI 消費裝置

產品定位
  • 本質就是 Smartphone
  • App 藏在 Agent 任務列後面
  • 晶片由高通、MTK 供應，每顆約 $120 USD
主持人態度：中性偏保守
  • 挑戰 iPhone 困難，挑 Pixel / Samsung 還有機會
  • 前車之鑑：AI Pin 走太前面 → BBQ
  • 夥伴是立訊，品質要直接打 iPhone 偏難
市場小作文已經把高通 / MTK 出貨量乘以 $120 喊到「正天價」——但別忘了這是血海戰場，中系、美系、韓系手機很快會跟上。

Meta × AWS Graviton5 多年合約

CPU 供不應求的證據
  • Andy Jassy 兩個月前法說：Graviton 自己用都不夠，一堆客戶要包下全部
  • Multi-billion $ × Tens of millions Graviton cores
  • 1,000 萬 cores ÷ 192 = 約 52K 顆晶片（且後面可能更多）

台股反應

  • 台系 CPU supply chain 全部動起來
  • 有趣的是「消費級 CPU chain」也動起來——市場在賭「排擠效應」劇本

「排擠效應」劇本能否複製？

記憶體經驗

  • 大家以為最缺的是 HBM / eSSD
  • 結果漲最兇的是 被排擠的 DDR3 / DDR4 / Nor / MLC NAND / 消費級 SSD

CPU 能否照抄？

支持理由

Intel 已明說產能優先給 Server，消費級會被排擠

主持人保守
  • 消費級終端需求不行
  • DDR3/4 本質仍是 Server 規格
  • 消費級 CPU 要像 D3/D4 那樣噴？「應該還好」
市場不少資金跑去買單這個故事；主持人沒買單，但保守看的東西「說不定後來是最噴的」——還是會持續觀察。
